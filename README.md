# Mini Task Manager

A full-stack task manager built with:

- Next.js
- Express.js
- TypeScript
- MongoDB
- Mongoose

## Features

- Create tasks
- View tasks
- Delete tasks
- Mark task as completed (work in progress)

## Run locally

### Client

```bash
cd client
npm install
npm run dev
```

### Server

```bash
cd server
npm install
npm run dev
```
